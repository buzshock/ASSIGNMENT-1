HTML Basics

(Hover Here to see content)

HTML (HyperText Markup Language) is the standard language for creating webpages. It forms the skeleton structure of web content.

Key HTML Elements

  • <html> - Root element
  • <head> - Contains meta-information about the document
  • <body> - Contains the visible content of the webpage

Common Tags

  • <h1>-<h6> - Header tags for titles
  • <p> - Paragraph tag
  • <a> - Anchor tag for links
  • <img> - Image tag for displaying images

CSS Basics

(Hover Here to see content)

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls the appearance of the elements defined in HTML.

Types of CSS

  • Inline CSS: Directly applied to HTML elements using the style attribute.
  • Internal CSS: Defined within a <style> tag in the <head> section.
  • External CSS: Linked to an HTML file using the <link> tag.

Common CSS Properties

  • color - Changes the text color
  • background-color - Sets the background color
  • font-size - Defines the size of text
  • margin and padding - Controls spacing between elements

Forms

(Hover Here to see content)

Forms are used to collect user input. HTML forms can contain various elements such as text fields, buttons, and checkboxes.

Common Form Elements

  • <input> - Defines an input field (e.g., text, email, password).
  • <textarea> - Defines a multi-line text input.
  • <button> - Defines a clickable button.

Input Types

  • text - Single-line text input
  • email - Input for email addresses, with automatic validation
  • password - Input for passwords, hides characters as typed
  • submit - A button that submits the form
